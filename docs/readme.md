# CRUD de Usuários
## Descrição
Este projeto é um sistema de gerenciamento de usuários simples, que permite adicionar, listar, editar e deletar usuários.

## Funcionalidades
* Adicionar usuários
* Listar usuários
* Editar usuários
* Deletar usuários
* Sair do sistema

## Tecnologias Utilizadas
* Python 3.x
* IPython (para limpar a tela)

## Estrutura do Projeto
O projeto é composto por um único arquivo Python, que contém todas as funcionalidades do sistema.

## Como Executar
Para executar o projeto, basta salvar o código em um arquivo Python (por exemplo, `crud.py`) e executá-lo usando o comando `python crud.py`.

## Exemplo de Uso
Ao executar o projeto, você será apresentado com um menu que permite escolher entre as seguintes opções:
